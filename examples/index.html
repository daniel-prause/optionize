This is an example page for the Optionize plugin.

Example 1: Adding Flags after option

You can add HTML elements before and after the option label.

Example:
<option value="germany" data-before-text="<img src='./images/germany.png'>" data-after-text="<div class='popup-opener'>.. hover here for more info</div><div class='popup'>is a country in Europe.</div>">
  Germany
</option>

Example 2: Adding Flags after option - with optgroups

You can add HTML elements before and after the optgroup-label as well.

Example:
<optgroup label="Europe" data-before-text="<strong>-</strong>" data-after-text="<strong>-</strong>">
  <option value="germany" data-before-text="<img src='./images/germany.png'>" data-after-text="<div class='popup-opener'>.. hover here for more info</div><div class='popup'>is a country in Europe.</div>">
    Germany
  </option>
</optgroup>
